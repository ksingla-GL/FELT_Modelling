FELT TOKEN FINANCIAL MODEL v1.0

HOW TO USE:
1. Edit input files in 'inputs' folder using Excel
2. Save all files as CSV (important!)
3. Double-click 'Run_Model.exe'
4. Check 'outputs' folder for results

INPUT FILES TO MODIFY:
- portfolio_config.csv: Farm costs, token parameters
- revenue_assumptions.csv: Production volumes by year
- cost_structure.csv: Stakeholder fee percentages
- carbon_pricing.csv: ACCU and environmental prices

OUTPUT FILES GENERATED:
- model_outputs.csv: Full 10-year projection
- summary_metrics.csv: Key performance indicators
- scenario_comparison.csv: Conservative/Base/Aggressive
- sensitivity_analysis.csv: Parameter impact testing
- token_metrics.csv: Token price progression
- stakeholder_distributions.csv: Fee allocations
